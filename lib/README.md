# CTRPluginFramework

**CTRPluginFramework** is a framework to build plugins for Nintendo 3DS games in *3GX* format.

This is a fork of [ThePixellizerOSS/CTRPluginFramework (GitLab)](https://gitlab.com/thepixellizeross/ctrpluginframework), with changes made specifically for this plugin.
